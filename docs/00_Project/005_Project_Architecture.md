# Project Architecture

## 文档体系架构

整个工程标准规范按**分层架构**组织：

```
┌─────────────────────────────────────────┐
│            README / 项目总览              │  ← 入口层
├─────────────────────────────────────────┤
│         00_Project  项目基础              │  ← 基础层
├─────────────────────────────────────────┤
│   01_Product    02_Domain               │  ← 业务层
│   产品设计        领域建模                │
├─────────────────────────────────────────┤
│   03_Architecture  04_Development        │  ← 工程层
│   架构设计          开发规范              │
├─────────────────────────────────────────┤
│   05_DevOps  06_Security  07_Operations  │  ← 基础设施层
│   CI/CD       安全         运维          │
├─────────────────────────────────────────┤
│     08_Compliance  09_Appendices         │  ← 治理层
│     合规            附录                 │
└─────────────────────────────────────────┘
```

## 工具链架构

```
┌───────────────────────────────────────────────┐
│                   MkDocs                       │  ← 文档站
│              (Material Theme)                   │
├───────────────────────────────────────────────┤
│                   GitHub                        │  ← 协作平台
│    Issues · PRs · Actions · Projects · Pages   │
├───────────────────────────────────────────────┤
│                templates/                       │  ← 模板引擎
│     Issue/PR/Doc/Code 模板 + Generator          │
├───────────────────────────────────────────────┤
│              checklists/                        │  ← 质量门禁
│     自动/手动检查清单 + CI 集成                  │
├───────────────────────────────────────────────┤
│               prompts/                          │  ← AI 辅助
│     AI Prompt 模板 + Contexts + Code Review     │
├───────────────────────────────────────────────┤
│               openapi/ · sql/                   │  ← 规范文件
│     API 规范 + 数据 Schema                      │
├───────────────────────────────────────────────┤
│               diagrams/ · adr/                  │  ← 决策记录
│     架构图 + 架构决策记录                        │
└───────────────────────────────────────────────┘
```
