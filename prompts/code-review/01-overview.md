# code-review / Overview

## 分类说明
code-review 分类下的 Prompt 模板列表。

## 模板列表

- 代码安全审查：检查 SQL 注入、XSS、CSRF 等安全漏洞
- 代码性能审查：识别性能瓶颈和优化机会
- 代码风格审查：检查是否符合项目编码规范
- API 设计审查：检查 RESTful API 设计是否符合规范
- 并发安全审查：检查线程安全和竞态条件
- 异常处理审查：检查异常处理是否完整正确
- 日志审查：检查日志记录是否规范和完整
- 依赖审查：检查第三方依赖的安全性和必要性
- 测试覆盖审查：检查测试是否覆盖核心逻辑
- 架构一致性审查：检查实现是否符合架构设计

## 使用方式
选择适合场景的模板，填入具体参数后使用。
