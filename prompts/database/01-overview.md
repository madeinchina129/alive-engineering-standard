# database / Overview

## 分类说明
database 分类下的 Prompt 模板列表。

## 模板列表

- 表结构设计：设计符合范式的关系表结构
- 索引策略设计：设计合理的索引策略
- 查询优化：优化复杂的 SQL 查询语句
- 数据迁移设计：设计安全的数据库迁移方案
- 分库分表设计：设计数据分片策略
- 缓存策略设计：设计多层缓存方案
- 备份恢复设计：设计数据库备份和恢复方案
- 数据归档设计：设计历史数据归档方案
- Sharding 设计：设计数据库分片键选择策略
- 读写分离设计：设计数据库读写分离架构

## 使用方式
选择适合场景的模板，填入具体参数后使用。
