# api-design / Overview

## 分类说明
api-design 分类下的 Prompt 模板列表。

## 模板列表

- RESTful API 设计：设计资源导向的 REST API
- GraphQL Schema 设计：设计 GraphQL 类型和查询
- API 版本策略：设计 API 版本管理方案
- 错误码设计：设计统一的 API 错误码体系
- API 分页设计：设计列表接口的分页方案
- API 鉴权设计：设计 API 认证和授权方案
- API 限流设计：设计 API 请求限流策略
- API 文档生成：自动生成 API 接口文档
- WebSocket 设计：设计实时通信 API
- gRPC API 设计：设计 gRPC 服务接口

## 使用方式
选择适合场景的模板，填入具体参数后使用。
