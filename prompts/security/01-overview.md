# security / Overview

## 分类说明
security 分类下的 Prompt 模板列表。

## 模板列表

- 安全漏洞扫描：扫描代码中的常见安全漏洞
- 认证方案设计：设计用户认证和会话管理方案
- 授权方案设计：设计细粒度的权限控制方案
- 数据加密方案：设计数据传输和存储加密方案
- 安全配置检查：检查安全相关配置是否正确
- API 安全设计：设计 API 的认证和限流方案
- 前端安全设计：设计前端 XSS/CSRF 防护方案
- 供应链安全：检查第三方依赖的安全性
- 合规性检查：检查是否符合合规要求
- 安全响应计划：设计安全事件的应急响应方案

## 使用方式
选择适合场景的模板，填入具体参数后使用。
