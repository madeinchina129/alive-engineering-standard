# documentation / Overview

## 分类说明
documentation 分类下的 Prompt 模板列表。

## 模板列表

- API 文档生成：根据代码生成 OpenAPI/Swagger 文档
- 架构文档生成：生成系统架构说明文档
- 部署文档生成：生成部署操作手册
- 用户手册生成：生成面向用户的使用说明
- 变更日志生成：根据提交记录生成版本变更日志
- 数据库设计文档：生成数据库表结构和关系文档
- 接口说明文档：生成服务间接口调用说明
- 故障处理文档：生成常见故障处理 SOP
- 开发环境搭建文档：生成开发环境配置指南
- API 迁移指南：生成 API 版本迁移说明文档

## 使用方式
选择适合场景的模板，填入具体参数后使用。
