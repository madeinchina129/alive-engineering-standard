# architecture / Overview

## 分类说明
architecture 分类下的 Prompt 模板列表。

## 模板列表

- 微服务拆分设计：根据业务域设计微服务边界
- 分层架构设计：设计应用的层级结构和职责划分
- 事件驱动架构：设计基于事件的系统通信方案
- CQRS 模式设计：设计命令和查询职责分离方案
- 领域驱动设计：设计限界上下文和聚合根
- API 网关设计：设计 API 网关的路由和治理方案
- 数据架构设计：设计数据存储和流转架构
- 安全架构设计：设计应用的整体安全架构
- 可观测性设计：设计日志、指标和追踪方案
- 灾备架构设计：设计高可用和灾备方案

## 使用方式
选择适合场景的模板，填入具体参数后使用。
