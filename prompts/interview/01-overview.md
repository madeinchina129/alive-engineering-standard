# interview / Overview

## 分类说明
interview 分类下的 Prompt 模板列表。

## 模板列表

- 技术面试题生成：生成 Java 核心技术面试题
- 系统设计面试：生成系统设计面试题目和解答
- 算法面试题：生成算法和数据结构面试题
- 前端面试题：生成前端技术面试题目
- 数据库面试题：生成数据库技术面试题
- 架构师面试题：生成架构师级别面试题目
- 项目管理面试题：生成项目管理面试题目
- DevOps 面试题：生成 DevOps 技术面试题
- 行为面试题：生成行为面试情景题
- Python 面试题：生成 Python 技术面试题

## 使用方式
选择适合场景的模板，填入具体参数后使用。
