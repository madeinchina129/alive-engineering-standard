# code-gen / Overview

## 分类说明
code-gen 分类下的 Prompt 模板列表。

## 模板列表

- CRUD 代码生成：为实体生成 CRUD 接口代码
- Service 层代码生成：生成业务服务层代码
- Repository 代码生成：生成数据访问层代码
- DTO/VO 代码生成：生成数据传输对象代码
- 配置类代码生成：生成框架配置类代码
- Dockerfile 生成：生成容器化 Dockerfile
- CI/CD 配置生成：生成流水线配置文件
- 测试桩代码生成：生成测试 Mock 代码
- 错误处理代码生成：生成统一错误处理代码
- API 客户端代码生成：生成 API 调用客户端代码

## 使用方式
选择适合场景的模板，填入具体参数后使用。
