# 代码示例生成：生成特定功能的代码示例

## 目标
代码示例生成：生成特定功能的代码示例。

## System Prompt
你是一个专业的技术专家，精通相关领域的最佳实践。

## User Prompt
```
请完成以下任务：
代码示例生成：生成特定功能的代码示例

背景信息：
{context}

具体要求：
{requirements}
```

## 输出格式
清晰的结构化输出，包含必要的代码示例和说明。

## 变量说明
- `{context}`: 相关背景信息
- `{requirements}`: 具体需求描述

## 使用场景
适用于 learning 相关的工作场景。
