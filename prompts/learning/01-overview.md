# learning / Overview

## 分类说明
learning 分类下的 Prompt 模板列表。

## 模板列表

- 技术概念解释：用简单语言解释复杂技术概念
- 代码教程生成：生成分步的代码教程
- 学习路径设计：设计技术学习路线图
- 知识总结：生成技术知识的结构化总结
- 实践练习生成：生成编程练习题目
- 技术对比分析：对比类似技术的优缺点
- 最佳实践总结：总结特定领域的最佳实践
- 常见错误总结：总结常见错误和解决方案
- 代码示例生成：生成特定功能的代码示例
- 读书笔记生成：生成技术书籍的读书笔记

## 使用方式
选择适合场景的模板，填入具体参数后使用。
