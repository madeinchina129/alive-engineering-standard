# testing / Overview

## 分类说明
testing 分类下的 Prompt 模板列表。

## 模板列表

- 单元测试生成：为 Service 层生成单元测试
- 集成测试生成：为 API 端点生成集成测试
- E2E 测试生成：生成用户路径的端到端测试
- 性能测试生成：生成压力测试和负载测试脚本
- 安全测试生成：生成安全渗透测试用例
- Mock 数据生成：为测试生成 Mock 数据和桩
- 边界测试生成：生成边界条件和异常场景测试
- UI 组件测试生成：为前端组件生成测试用例
- 数据库测试生成：生成数据访问层测试
- 回归测试选择：根据代码变更推荐回归测试范围

## 使用方式
选择适合场景的模板，填入具体参数后使用。
