# refactoring / Overview

## 分类说明
refactoring 分类下的 Prompt 模板列表。

## 模板列表

- 代码重构建议：识别代码坏味道并推荐重构方案
- 函数拆分建议：推荐将大函数拆分为小函数
- 设计模式引入：推荐适合当前代码的设计模式
- API 重构建议：推荐 API 接口的优化方案
- 数据库重构：推荐数据库 Schema 的优化方案
- 模块解耦建议：推荐模块间的解耦方案
- 遗留系统现代化：推荐老系统的现代化改造方案
- 性能重构建议：推荐针对性能的代码重构方案
- 测试代码重构：推荐测试代码的优化改进
- 配置重构建议：推荐配置管理的优化方案

## 使用方式
选择适合场景的模板，填入具体参数后使用。
