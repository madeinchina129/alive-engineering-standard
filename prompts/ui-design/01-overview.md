# ui-design / Overview

## 分类说明
ui-design 分类下的 Prompt 模板列表。

## 模板列表

- 组件设计：设计可复用的 UI 组件
- 页面布局设计：设计应用的整体页面布局
- 表单设计：设计高效的用户输入表单
- 数据可视化设计：设计数据图表展示方案
- 移动端适配设计：设计移动端 UI 适配方案
- 暗黑模式设计：设计深色主题方案
- 动效设计：设计合理的界面动效方案
- 无障碍设计：设计符合无障碍标准的界面
- 设计系统构建：构建设计系统和组件库
- 设计评审：审查 UI 设计稿的规范一致性

## 使用方式
选择适合场景的模板，填入具体参数后使用。
