# Context Documents

AI 上下文文档目录。用于在 AI 编码时提供领域上下文，确保 AI 理解项目规范。

## 文件清单

| 文件 | 说明 |
|------|------|
| `project.md` | 项目全局上下文 |
| `flutter.md` | Flutter 开发上下文 |
| `vue3.md` | Vue3 开发上下文 |
| `springboot.md` | Spring Boot 开发上下文 |
| `ai.md` | AI 开发上下文 |

## 使用方式

```bash
# 在编码时加载对应上下文，让 AI 了解项目规范
cat contexts/flutter.md
```
