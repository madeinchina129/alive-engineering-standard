# Contributing

感谢你考虑为本项目做出贡献！

## 提交 Issue

- 使用提供的模板提交 Bug 报告或功能请求
- 清晰描述问题或建议

## 提交 Pull Request

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/amazing-feature`)
3. 提交你的改动 (`git commit -m 'feat: add amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 提交 Pull Request

## 开发规范

请参考项目中的标准文档和模板。
