# 合规风险管理 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| CMP-RM-001 | 合规风险按照发生可能性和影响程度分为高/中/低三级 | P0 | 是 |
| CMP-RM-002 | 高风险合规项必须每月跟踪状态，中风险每季度 | P0 | 是 |
| CMP-RM-003 | 合规风险登记册必须包含：风险描述、影响分析、整改措施、责任人、截止日期 | P0 | 是 |
| CMP-RM-004 | 发生合规事件后必须在 24 小时内上报合规委员会 | P0 | 是 |

## 详细说明

### CMP-RM-001（P0）
合规风险按照发生可能性和影响程度分为高/中/低三级

### CMP-RM-002（P0）
高风险合规项必须每月跟踪状态，中风险每季度

### CMP-RM-003（P0）
合规风险登记册必须包含：风险描述、影响分析、整改措施、责任人、截止日期

### CMP-RM-004（P0）
发生合规事件后必须在 24 小时内上报合规委员会
