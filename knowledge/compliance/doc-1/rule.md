# 合规要求管理 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| CMP-001 | 所有业务条线必须建立合规要求清单，至少每半年更新一次 | P0 | 是 |
| CMP-002 | 合规要求必须标注：来源、适用范围、风险等级、合规截止日期 | P0 | 是 |
| CMP-003 | 高风险合规要求必须有独立验证机制 | P0 | 是 |
| CMP-004 | 合规差距必须在发现后 30 天内制定整改计划 | P0 | 是 |

## 详细说明

### CMP-001（P0）
所有业务条线必须建立合规要求清单，至少每半年更新一次

### CMP-002（P0）
合规要求必须标注：来源、适用范围、风险等级、合规截止日期

### CMP-003（P0）
高风险合规要求必须有独立验证机制

### CMP-004（P0）
合规差距必须在发现后 30 天内制定整改计划
