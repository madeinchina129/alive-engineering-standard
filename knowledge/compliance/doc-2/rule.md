# 审计追踪规范 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| CMP-AUD-001 | 以下操作必须记录审计日志：用户认证、权限变更、数据修改、敏感数据访问 | P0 | 是 |
| CMP-AUD-002 | 每一条审计日志必须包含：时间戳、操作人、操作类型、操作对象、操作结果、源 IP | P0 | 是 |
| CMP-AUD-003 | 审计日志存储周期不少于 180 天，金融类业务不少于 365 天 | P0 | 是 |
| CMP-AUD-004 | 审计日志需定期归档，归档后不得修改 | P0 | 是 |

## 详细说明

### CMP-AUD-001（P0）
以下操作必须记录审计日志：用户认证、权限变更、数据修改、敏感数据访问

### CMP-AUD-002（P0）
每一条审计日志必须包含：时间戳、操作人、操作类型、操作对象、操作结果、源 IP

### CMP-AUD-003（P0）
审计日志存储周期不少于 180 天，金融类业务不少于 365 天

### CMP-AUD-004（P0）
审计日志需定期归档，归档后不得修改
