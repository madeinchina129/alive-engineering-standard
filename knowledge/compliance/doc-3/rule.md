# 数据隐私合规 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| CMP-PRI-001 | 所有涉及个人信息的系统必须完成隐私影响评估（PIA） | P0 | 是 |
| CMP-PRI-002 | 个人信息收集前必须获得用户的明确同意（Opt-in） | P0 | 是 |
| CMP-PRI-003 | 用户数据删除请求必须在 15 个工作日内响应 | P0 | 是 |
| CMP-PRI-004 | 个人敏感信息（生物识别、金融账户、健康数据）必须加密存储 | P0 | 是 |
| CMP-PRI-005 | 数据跨境传输必须通过安全评估 | P0 | 是 |

## 详细说明

### CMP-PRI-001（P0）
所有涉及个人信息的系统必须完成隐私影响评估（PIA）

### CMP-PRI-002（P0）
个人信息收集前必须获得用户的明确同意（Opt-in）

### CMP-PRI-003（P0）
用户数据删除请求必须在 15 个工作日内响应

### CMP-PRI-004（P0）
个人敏感信息（生物识别、金融账户、健康数据）必须加密存储

### CMP-PRI-005（P0）
数据跨境传输必须通过安全评估
