# GDPR 合规指南 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| CMP-GDPR-001 | 处理欧盟用户数据的业务必须指定欧盟代表 | P0 | 是 |
| CMP-GDPR-002 | 数据泄露必须在 72 小时内通知监管机构 | P0 | 是 |
| CMP-GDPR-003 | 数据处理必须有合法依据（同意/合同/法律义务/正当利益） | P0 | 是 |
| CMP-GDPR-004 | 用户有权要求数据可移植性（以通用格式导出个人数据） | P0 | 是 |
| CMP-GDPR-005 | DPO（数据保护官）任命必须在涉及大规模监控或敏感数据处理时完成 | P0 | 是 |

## 详细说明

### CMP-GDPR-001（P0）
处理欧盟用户数据的业务必须指定欧盟代表

### CMP-GDPR-002（P0）
数据泄露必须在 72 小时内通知监管机构

### CMP-GDPR-003（P0）
数据处理必须有合法依据（同意/合同/法律义务/正当利益）

### CMP-GDPR-004（P0）
用户有权要求数据可移植性（以通用格式导出个人数据）

### CMP-GDPR-005（P0）
DPO（数据保护官）任命必须在涉及大规模监控或敏感数据处理时完成
