# 基准测试规范 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| PERF-BENCH-001 | 基准测试必须在与生产环境配置一致的隔离环境中执行 | P0 | 是 |
| PERF-BENCH-002 | 测试数据量需达到生产环境的 50% 以上 | P0 | 是 |
| PERF-BENCH-003 | 每次基准测试至少运行 3 次取中位数 | P0 | 是 |
| PERF-BENCH-004 | 基准测试结果需与上一次基准测试结果对比，偏差超过 15% 需分析原因 | P0 | 是 |
| PERF-BENCH-005 | 基准测试脚本和数据版本化管理 | P1 | 推荐 |

## 详细说明

### PERF-BENCH-001（P0）
基准测试必须在与生产环境配置一致的隔离环境中执行

### PERF-BENCH-002（P0）
测试数据量需达到生产环境的 50% 以上

### PERF-BENCH-003（P0）
每次基准测试至少运行 3 次取中位数

### PERF-BENCH-004（P0）
基准测试结果需与上一次基准测试结果对比，偏差超过 15% 需分析原因

### PERF-BENCH-005（P1）
基准测试脚本和数据版本化管理
