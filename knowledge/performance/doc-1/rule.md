# 性能指标体系 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| PERF-MET-001 | 性能指标分为三类：用户体验指标、系统指标、资源指标 | P0 | 是 |
| PERF-MET-002 | 用户体验指标必须包含：LCP、FID/INP、CLS（Web 核心指标） | P0 | 是 |
| PERF-MET-003 | 系统指标必须包含：P50/P95/P99 延迟、吞吐量、错误率 | P0 | 是 |
| PERF-MET-004 | 所有性能指标必须有明确的采集方式和计算口径 | P0 | 是 |
| PERF-MET-005 | 性能基线每月更新一次，反映系统性能趋势 | P1 | 推荐 |

## 详细说明

### PERF-MET-001（P0）
性能指标分为三类：用户体验指标、系统指标、资源指标

### PERF-MET-002（P0）
用户体验指标必须包含：LCP、FID/INP、CLS（Web 核心指标）

### PERF-MET-003（P0）
系统指标必须包含：P50/P95/P99 延迟、吞吐量、错误率

### PERF-MET-004（P0）
所有性能指标必须有明确的采集方式和计算口径

### PERF-MET-005（P1）
性能基线每月更新一次，反映系统性能趋势
