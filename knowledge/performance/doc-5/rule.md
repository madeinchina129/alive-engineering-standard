# 负载测试标准 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| PERF-LOAD-001 | 每次重大版本发布前必须完成负载测试 | P0 | 是 |
| PERF-LOAD-002 | 负载测试必须包含：基准测试、负载测试、压力测试、耐久测试 | P0 | 是 |
| PERF-LOAD-003 | 测试负载模型必须基于生产环境的真实流量数据 | P0 | 是 |
| PERF-LOAD-004 | 测试期间必须监控：响应时间、吞吐量、错误率、资源使用率 | P0 | 是 |
| PERF-LOAD-005 | 测试结果必须包含：系统瓶颈点、最大安全吞吐量、建议限流阈值 | P0 | 是 |

## 详细说明

### PERF-LOAD-001（P0）
每次重大版本发布前必须完成负载测试

### PERF-LOAD-002（P0）
负载测试必须包含：基准测试、负载测试、压力测试、耐久测试

### PERF-LOAD-003（P0）
测试负载模型必须基于生产环境的真实流量数据

### PERF-LOAD-004（P0）
测试期间必须监控：响应时间、吞吐量、错误率、资源使用率

### PERF-LOAD-005（P0）
测试结果必须包含：系统瓶颈点、最大安全吞吐量、建议限流阈值
