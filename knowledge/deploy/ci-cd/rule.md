# CI/CD 流水线标准 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| CI-001 | 每次代码推送触发 CI 流水线 | P0 | 是 |
| CI-002 | CI 必须包含：lint → 单元测试 → 构建 → 集成测试 | P0 | 是 |
| CI-003 | 部署到生产环境需要审批 | P1 | 是 |

## 详细说明

### CI-001（P0）
每次代码推送触发 CI 流水线

### CI-002（P0）
CI 必须包含：lint → 单元测试 → 构建 → 集成测试

### CI-003（P1）
部署到生产环境需要审批
