# 环境管理规范 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| DEP-ENV-001 | 所有环境使用 IaC（Terraform/Pulumi）管理，禁止手动创建资源 | P0 | 是 |
| DEP-ENV-002 | 生产环境和预发布环境的差异最小化（只调整规格和密钥） | P0 | 是 |
| DEP-ENV-003 | 生产环境禁止直接部署代码，必须通过 CI/CD 流水线 | P0 | 是 |
| DEP-ENV-004 | 生产环境数据库仅通过自动化脚本访问，禁止直接 SQL 操作 | P0 | 是 |
| DEP-ENV-005 | 测试环境使用脱敏数据（不能包含真实用户个人信息） | P0 | 是 |
| DEP-ENV-006 | 环境配置（环境变量/Feature Flag）集中在统一配置中心管理 | P1 | 是 |

## 详细说明

### DEP-ENV-001（P0）
所有环境使用 IaC（Terraform/Pulumi）管理，禁止手动创建资源

### DEP-ENV-002（P0）
生产环境和预发布环境的差异最小化（只调整规格和密钥）

### DEP-ENV-003（P0）
生产环境禁止直接部署代码，必须通过 CI/CD 流水线

### DEP-ENV-004（P0）
生产环境数据库仅通过自动化脚本访问，禁止直接 SQL 操作

### DEP-ENV-005（P0）
测试环境使用脱敏数据（不能包含真实用户个人信息）

### DEP-ENV-006（P1）
环境配置（环境变量/Feature Flag）集中在统一配置中心管理
