# 模型训练流水线 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| AI-TP-001 | 训练数据集必须有质量门禁（缺失率、异常值、分布检查） | P0 | 是 |
| AI-TP-002 | 训练前必须将数据集划分为训练/验证/测试（80/10/10） | P0 | 是 |
| AI-TP-003 | 分布式训练必须使用混合精度训练（FP16）提升效率 | P0 | 是 |
| AI-TP-004 | 超参搜索至少使用贝叶斯优化，禁止手动调参 | P0 | 是 |
| AI-TP-005 | 训练过程必须记录损失曲线和学习率变化 | P0 | 是 |
| AI-TP-006 | Early Stopping 启用，连续 N 个 epoch 验证集无改善时停止 | P0 | 是 |

## 详细说明

### AI-TP-001（P0）
训练数据集必须有质量门禁（缺失率、异常值、分布检查）

### AI-TP-002（P0）
训练前必须将数据集划分为训练/验证/测试（80/10/10）

### AI-TP-003（P0）
分布式训练必须使用混合精度训练（FP16）提升效率

### AI-TP-004（P0）
超参搜索至少使用贝叶斯优化，禁止手动调参

### AI-TP-005（P0）
训练过程必须记录损失曲线和学习率变化

### AI-TP-006（P0）
Early Stopping 启用，连续 N 个 epoch 验证集无改善时停止
