# MLOps 规范 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| AI-MLOPS-001 | 所有实验必须使用实验跟踪工具（MLflow/Weights & Biases）记录 | P0 | 是 |
| AI-MLOPS-002 | 模型训练代码、数据和配置必须版本化管理（DVC + Git） | P0 | 是 |
| AI-MLOPS-003 | 模型晋升流程：开发 → Staging（离线评估）→ Production（A/B 评估） | P0 | 是 |
| AI-MLOPS-004 | 生产环境模型必须有自动回滚机制（监控指标退化自动触发回滚） | P0 | 是 |
| AI-MLOPS-005 | 模型服务必须有资源使用监控（GPU/CPU 利用率、推理延迟） | P0 | 是 |
| AI-MLOPS-006 | 数据漂移检测每日运行，触发告警后进行模型重训练评估 | P1 | 推荐 |

## 详细说明

### AI-MLOPS-001（P0）
所有实验必须使用实验跟踪工具（MLflow/Weights & Biases）记录

### AI-MLOPS-002（P0）
模型训练代码、数据和配置必须版本化管理（DVC + Git）

### AI-MLOPS-003（P0）
模型晋升流程：开发 → Staging（离线评估）→ Production（A/B 评估）

### AI-MLOPS-004（P0）
生产环境模型必须有自动回滚机制（监控指标退化自动触发回滚）

### AI-MLOPS-005（P0）
模型服务必须有资源使用监控（GPU/CPU 利用率、推理延迟）

### AI-MLOPS-006（P1）
数据漂移检测每日运行，触发告警后进行模型重训练评估
