# LLM 集成规范 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| LLM-001 | 所有 LLM 调用通过统一的 AI Service 层转发 | P0 | 是 |
| LLM-002 | 必须实现超时和重试机制 | P1 | 是 |
| LLM-003 | 调用日志记录模型、Token 和延迟 | P1 | 是 |

## 详细说明

### LLM-001（P0）
所有 LLM 调用通过统一的 AI Service 层转发

### LLM-002（P1）
必须实现超时和重试机制

### LLM-003（P1）
调用日志记录模型、Token 和延迟
