# 设计令牌系统 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| DT-001 | 所有颜色值使用语义化 Token 而非直接色值 | P0 | 是 |
| DT-002 | 间距单位使用 4px 基准网格 | P1 | 是 |
| DT-003 | 字号使用比例尺系统（1.25 Major Third） | P1 | 是 |

## 详细说明

### DT-001（P0）
所有颜色值使用语义化 Token 而非直接色值

### DT-002（P1）
间距单位使用 4px 基准网格

### DT-003（P1）
字号使用比例尺系统（1.25 Major Third）
