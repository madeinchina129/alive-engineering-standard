# 无障碍扩展实践 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| UI-A11X-001 | 复杂组件必须提供完整的 ARIA 角色和属性 | P0 | 是 |
| UI-A11X-002 | 数据表格需提供行/列标题关联和排序状态通知 | P0 | 是 |
| UI-A11X-003 | 拖拽操作必须提供键盘替代方案（重新排序按钮） | P0 | 是 |
| UI-A11X-004 | 弹窗/抽屉/菜单等浮层组件需管理焦点陷阱和 ESC 关闭 | P0 | 是 |
| UI-A11X-005 | 路由切换时需将焦点移到页面顶部并通知标题变化 | P1 | 是 |
| UI-A11X-006 | 超过 3 秒的加载过程需提供进度指示器和状态描述 | P1 | 是 |

## 详细说明

### UI-A11X-001（P0）
复杂组件必须提供完整的 ARIA 角色和属性

### UI-A11X-002（P0）
数据表格需提供行/列标题关联和排序状态通知

### UI-A11X-003（P0）
拖拽操作必须提供键盘替代方案（重新排序按钮）

### UI-A11X-004（P0）
弹窗/抽屉/菜单等浮层组件需管理焦点陷阱和 ESC 关闭

### UI-A11X-005（P1）
路由切换时需将焦点移到页面顶部并通知标题变化

### UI-A11X-006（P1）
超过 3 秒的加载过程需提供进度指示器和状态描述
