# 无障碍访问规范 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| UI-A11Y-001 | 所有图片必须有 alt 文本描述 | P0 | 是 |
| UI-A11Y-002 | 颜色对比度至少达到 AA 级（文本 4.5:1，大文本 3:1） | P0 | 是 |
| UI-A11Y-003 | 所有交互元素必须可通过键盘 Tab 键聚焦和操作 | P0 | 是 |
| UI-A11Y-004 | 焦点指示器必须清晰可见（outline 样式，最小 2px） | P0 | 是 |
| UI-A11Y-005 | 表单输入必须有关联的 label 标签 | P0 | 是 |
| UI-A11Y-006 | 错误提示信息需通过 aria-describedby 关联到输入框 | P1 | 是 |
| UI-A11Y-007 | 动态内容更新需使用 aria-live region 通知屏幕阅读器 | P1 | 是 |
| UI-A11Y-008 | 页面 landmarks（header/nav/main/footer）需正确使用语义化标签 | P1 | 是 |

## 详细说明

### UI-A11Y-001（P0）
所有图片必须有 alt 文本描述

### UI-A11Y-002（P0）
颜色对比度至少达到 AA 级（文本 4.5:1，大文本 3:1）

### UI-A11Y-003（P0）
所有交互元素必须可通过键盘 Tab 键聚焦和操作

### UI-A11Y-004（P0）
焦点指示器必须清晰可见（outline 样式，最小 2px）

### UI-A11Y-005（P0）
表单输入必须有关联的 label 标签

### UI-A11Y-006（P1）
错误提示信息需通过 aria-describedby 关联到输入框

### UI-A11Y-007（P1）
动态内容更新需使用 aria-live region 通知屏幕阅读器

### UI-A11Y-008（P1）
页面 landmarks（header/nav/main/footer）需正确使用语义化标签
