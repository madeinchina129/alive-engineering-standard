# 无障碍访问规范 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| A11Y-001 | 所有图片必须有 alt 文本 | P0 | 是 |
| A11Y-002 | 所有可交互元素必须可键盘访问 | P0 | 是 |
| A11Y-003 | 颜色对比度至少达到 WCAG AA 标准 | P1 | 是 |

## 详细说明

### A11Y-001（P0）
所有图片必须有 alt 文本

### A11Y-002（P0）
所有可交互元素必须可键盘访问

### A11Y-003（P1）
颜色对比度至少达到 WCAG AA 标准
