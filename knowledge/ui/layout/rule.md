# 布局系统规范 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| UI-LAY-001 | 所有间距使用 8px 基准网格（spacing-4/8/12/16/24/32/48/64） | P0 | 是 |
| UI-LAY-002 | 页面内容区域最多 3 层嵌套：Page > Section > Card | P0 | 是 |
| UI-LAY-003 | 页面左右保留至少 16px（移动端）/ 24px（桌面端）安全间距 | P0 | 是 |
| UI-LAY-004 | 文案行宽不超过 720px（约 40-70 字符/行），保证可读性 | P1 | 推荐 |
| UI-LAY-005 | 布局使用 CSS Grid + Flexbox，避免使用 float 布局 | P0 | 是 |

## 详细说明

### UI-LAY-001（P0）
所有间距使用 8px 基准网格（spacing-4/8/12/16/24/32/48/64）

### UI-LAY-002（P0）
页面内容区域最多 3 层嵌套：Page > Section > Card

### UI-LAY-003（P0）
页面左右保留至少 16px（移动端）/ 24px（桌面端）安全间距

### UI-LAY-004（P1）
文案行宽不超过 720px（约 40-70 字符/行），保证可读性

### UI-LAY-005（P0）
布局使用 CSS Grid + Flexbox，避免使用 float 布局
