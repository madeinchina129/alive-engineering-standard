# 主题与配色方案 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| UI-THM-001 | 所有颜色值必须通过主题变量引用，颜色语义化命名（color/text-primary） | P0 | 是 |
| UI-THM-002 | 主题定义必须包含：浅色模式、深色模式两套完整配置 | P0 | 是 |
| UI-THM-003 | 主题切换使用 CSS Custom Properties 方案（非 JS 运行时注入） | P0 | 是 |
| UI-THM-004 | 深色模式非简单反色，需重新设计阴影深度和背景层次 | P1 | 是 |
| UI-THM-005 | 品牌色表至少包含 10 个阶度（50-900） | P1 | 推荐 |

## 详细说明

### UI-THM-001（P0）
所有颜色值必须通过主题变量引用，颜色语义化命名（color/text-primary）

### UI-THM-002（P0）
主题定义必须包含：浅色模式、深色模式两套完整配置

### UI-THM-003（P0）
主题切换使用 CSS Custom Properties 方案（非 JS 运行时注入）

### UI-THM-004（P1）
深色模式非简单反色，需重新设计阴影深度和背景层次

### UI-THM-005（P1）
品牌色表至少包含 10 个阶度（50-900）
