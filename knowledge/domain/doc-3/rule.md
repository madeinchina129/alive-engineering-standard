# 聚合设计规范 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| DDD-AGG-001 | 聚合必须通过聚合根访问，外部不能直接操作聚合内部实体 | P0 | 是 |
| DDD-AGG-002 | 一个事务只能修改一个聚合实例 | P0 | 是 |
| DDD-AGG-003 | 聚合根必须使用全局唯一标识 | P0 | 是 |
| DDD-AGG-004 | 聚合内实体不超过 5 个，超过应考虑拆分 | P1 | 推荐 |
| DDD-AGG-005 | 聚合根必须保证不变量（Invariants）的完整性 | P0 | 是 |

## 详细说明

### DDD-AGG-001（P0）
聚合必须通过聚合根访问，外部不能直接操作聚合内部实体

### DDD-AGG-002（P0）
一个事务只能修改一个聚合实例

### DDD-AGG-003（P0）
聚合根必须使用全局唯一标识

### DDD-AGG-004（P1）
聚合内实体不超过 5 个，超过应考虑拆分

### DDD-AGG-005（P0）
聚合根必须保证不变量（Invariants）的完整性
