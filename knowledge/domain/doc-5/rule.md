# 值对象实现 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| DDD-VO-001 | 值对象必须为不可变（所有属性在构造后不可修改） | P0 | 是 |
| DDD-VO-002 | 值对象必须实现 equals 和 hashCode 方法（基于值的比较） | P0 | 是 |
| DDD-VO-003 | 值对象应包含自验证逻辑，不依赖外部进行校验 | P0 | 是 |
| DDD-VO-004 | 值对象可与实体关联，但不能取代实体的身份 | P1 | 推荐 |

## 详细说明

### DDD-VO-001（P0）
值对象必须为不可变（所有属性在构造后不可修改）

### DDD-VO-002（P0）
值对象必须实现 equals 和 hashCode 方法（基于值的比较）

### DDD-VO-003（P0）
值对象应包含自验证逻辑，不依赖外部进行校验

### DDD-VO-004（P1）
值对象可与实体关联，但不能取代实体的身份
