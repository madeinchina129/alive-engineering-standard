# 通用语言定义 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| DDD-UL-001 | 每个限界上下文必须有专属的术语表（Glossary） | P0 | 是 |
| DDD-UL-002 | 术语定义必须包含：名称、定义、同义词、使用上下文 | P0 | 是 |
| DDD-UL-003 | 不同上下文中的同名术语必须在术语表中注明歧义 | P0 | 是 |
| DDD-UL-004 | 代码中的类名和方法名必须与术语表保持一致 | P1 | 是 |

## 详细说明

### DDD-UL-001（P0）
每个限界上下文必须有专属的术语表（Glossary）

### DDD-UL-002（P0）
术语定义必须包含：名称、定义、同义词、使用上下文

### DDD-UL-003（P0）
不同上下文中的同名术语必须在术语表中注明歧义

### DDD-UL-004（P1）
代码中的类名和方法名必须与术语表保持一致
