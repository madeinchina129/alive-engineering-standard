# Rust 所有权 Checklist

## 所有权
- [ ] 是否理解每个值只有一个所有者？
- [ ] 是否避免不必要的所有权转移？
- [ ] 函数参数是否优先使用引用？
- [ ] 返回值是否避免返回局部变量引用？

## 借用
- [ ] 不可变引用是否不用于可变引用同时存在？
- [ ] 是否避免同时持有两个可变引用？
- [ ] 是否使用 `&str` 而非 `&String` 作为参数？
- [ ] 借用是否在最后一次使用后结束？

## 生命周期
- [ ] 函数返回引用时标注了生命周期？
- [ ] 结构体包含引用时标注了生命周期？
- [ ] impl 块是否标注了生命周期？

## 智能指针
- [ ] 单线程共享所有权是否使用 `Rc<T>`？
- [ ] 多线程共享所有权是否使用 `Arc<T>`？
- [ ] 运行时借用检查是否使用 `RefCell<T>`？
- [ ] trait object 是否使用 `Box<dyn Trait>`？

## 常用检查
- [ ] `cargo clippy` 是否零 warning？
- [ ] `cargo check` 是否通过？
- [ ] 是否避免不必要的 `clone()`？
- [ ] 是否避免使用 `unsafe` 绕过借用检查？
