# 市场研究方法 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| MR-001 | 重要产品决策（影响 20%+ 用户）必须经过用户研究验证 | P0 | 是 |
| MR-002 | 用户访谈样本量不少于 8 人，问卷调查不少于 200 份有效回收 | P0 | 是 |
| MR-003 | 研究报告必须包含：研究目的、方法、发现、建议、附录 | P0 | 是 |
| MR-004 | 定量数据需做统计显著性检验（p < 0.05） | P1 | 是 |
| MR-005 | 次级数据（行业报告、竞品数据）需标注来源和时效性 | P1 | 是 |

## 详细说明

### MR-001（P0）
重要产品决策（影响 20%+ 用户）必须经过用户研究验证

### MR-002（P0）
用户访谈样本量不少于 8 人，问卷调查不少于 200 份有效回收

### MR-003（P0）
研究报告必须包含：研究目的、方法、发现、建议、附录

### MR-004（P1）
定量数据需做统计显著性检验（p < 0.05）

### MR-005（P1）
次级数据（行业报告、竞品数据）需标注来源和时效性
