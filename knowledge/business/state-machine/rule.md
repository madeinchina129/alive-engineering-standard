# KPI 指标体系 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| KPI-001 | 每个产品线必须有 1 个北极星指标（North Star Metric） | P0 | 是 |
| KPI-002 | 一级指标不超过 5 个，二级指标不超过 15 个 | P0 | 是 |
| KPI-003 | 所有指标必须有数据口径定义文档，包含数据来源和计算逻辑 | P0 | 是 |
| KPI-004 | 指标体系每月回顾一次，评估指标的有效性和准确性 | P1 | 推荐 |
| KPI-005 | 仪表盘数据刷新延迟不超过 T+1 | P1 | 是 |

## 详细说明

### KPI-001（P0）
每个产品线必须有 1 个北极星指标（North Star Metric）

### KPI-002（P0）
一级指标不超过 5 个，二级指标不超过 15 个

### KPI-003（P0）
所有指标必须有数据口径定义文档，包含数据来源和计算逻辑

### KPI-004（P1）
指标体系每月回顾一次，评估指标的有效性和准确性

### KPI-005（P1）
仪表盘数据刷新延迟不超过 T+1
