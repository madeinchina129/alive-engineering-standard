# 业务战略规划 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| BS-001 | 每季度进行一次战略回顾，更新 OKR 和目标 | P0 | 是 |
| BS-002 | 新业务方向必须有 MVP 验证计划和成功/失败标准 | P0 | 是 |
| BS-003 | 战略目标使用 OKR 框架分解到每个团队和个人 | P0 | 是 |
| BS-004 | 竞品分析每季度更新一次，覆盖直接和间接竞品 | P1 | 推荐 |
| BS-005 | 市场趋势分析（PEST）每年至少更新一次 | P1 | 推荐 |
| BS-006 | 战略变更需经管理团队评审并通知所有相关方 | P1 | 是 |

## 详细说明

### BS-001（P0）
每季度进行一次战略回顾，更新 OKR 和目标

### BS-002（P0）
新业务方向必须有 MVP 验证计划和成功/失败标准

### BS-003（P0）
战略目标使用 OKR 框架分解到每个团队和个人

### BS-004（P1）
竞品分析每季度更新一次，覆盖直接和间接竞品

### BS-005（P1）
市场趋势分析（PEST）每年至少更新一次

### BS-006（P1）
战略变更需经管理团队评审并通知所有相关方
