# 指标定义规范 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| ANL-MET-001 | 指标命名采用 业务域_指标名 格式（如 order_gmv, user_retention_7d） | P0 | 是 |
| ANL-MET-002 | 每个指标必须明确：口径说明、计算逻辑、数据源、更新频率 | P0 | 是 |
| ANL-MET-003 | 指标变更必须通过评审，更新版本号并通知所有消费者 | P0 | 是 |
| ANL-MET-004 | 比率类指标必须注明分子和分母的口径 | P0 | 是 |
| ANL-MET-005 | 指标看板必须标注数据延迟时间 | P1 | 推荐 |

## 详细说明

### ANL-MET-001（P0）
指标命名采用 业务域_指标名 格式（如 order_gmv, user_retention_7d）

### ANL-MET-002（P0）
每个指标必须明确：口径说明、计算逻辑、数据源、更新频率

### ANL-MET-003（P0）
指标变更必须通过评审，更新版本号并通知所有消费者

### ANL-MET-004（P0）
比率类指标必须注明分子和分母的口径

### ANL-MET-005（P1）
指标看板必须标注数据延迟时间
