# 数据可视化标准 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| ANL-VIZ-001 | 趋势数据使用折线图，对比数据使用柱状图，占比数据使用饼图/环形图 | P0 | 是 |
| ANL-VIZ-002 | 颜色使用不超过 7 种，且需考虑色盲友好（避免红绿搭配） | P0 | 是 |
| ANL-VIZ-003 | 图表必须标注坐标轴说明、数据单位和数据更新时间 | P0 | 是 |
| ANL-VIZ-004 | 指标卡必须显示：当前值、变化趋势、变化百分比、对比基准 | P0 | 是 |
| ANL-VIZ-005 | 看板加载时间不得超过 3 秒，大数据量需预聚合 | P1 | 推荐 |

## 详细说明

### ANL-VIZ-001（P0）
趋势数据使用折线图，对比数据使用柱状图，占比数据使用饼图/环形图

### ANL-VIZ-002（P0）
颜色使用不超过 7 种，且需考虑色盲友好（避免红绿搭配）

### ANL-VIZ-003（P0）
图表必须标注坐标轴说明、数据单位和数据更新时间

### ANL-VIZ-004（P0）
指标卡必须显示：当前值、变化趋势、变化百分比、对比基准

### ANL-VIZ-005（P1）
看板加载时间不得超过 3 秒，大数据量需预聚合
