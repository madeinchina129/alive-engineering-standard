# A/B 测试规范 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| ANL-AB-001 | A/B 测试必须提前定义：实验假设、核心指标、最小 detectable effect | P0 | 是 |
| ANL-AB-002 | 分流必须基于 user_id 哈希，确保用户在不同实验间的一致性 | P0 | 是 |
| ANL-AB-003 | 实验最小运行周期为 7 天，覆盖完整的周周期效应 | P0 | 是 |
| ANL-AB-004 | 统计显著性水平设为 95%（p < 0.05），严禁提前停止实验 | P0 | 是 |
| ANL-AB-005 | 同时运行的互斥实验不能超过 3 个，避免流量竞争 | P1 | 推荐 |

## 详细说明

### ANL-AB-001（P0）
A/B 测试必须提前定义：实验假设、核心指标、最小 detectable effect

### ANL-AB-002（P0）
分流必须基于 user_id 哈希，确保用户在不同实验间的一致性

### ANL-AB-003（P0）
实验最小运行周期为 7 天，覆盖完整的周周期效应

### ANL-AB-004（P0）
统计显著性水平设为 95%（p < 0.05），严禁提前停止实验

### ANL-AB-005（P1）
同时运行的互斥实验不能超过 3 个，避免流量竞争
