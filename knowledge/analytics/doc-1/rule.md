# 数据分析规范 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| ANL-001 | 每次分析必须明确：分析目标、数据来源、分析方法、预期产出 | P0 | 是 |
| ANL-002 | 分析报告必须包含：摘要、数据说明、分析方法、结果、结论与建议 | P0 | 是 |
| ANL-003 | 数据清洗过程必须记录去除的数据量、原因和过滤条件 | P0 | 是 |
| ANL-004 | 统计分析必须注明置信区间和显著性水平 | P1 | 推荐 |
| ANL-005 | 分析结果必须经过至少一人交叉验证 | P1 | 推荐 |

## 详细说明

### ANL-001（P0）
每次分析必须明确：分析目标、数据来源、分析方法、预期产出

### ANL-002（P0）
分析报告必须包含：摘要、数据说明、分析方法、结果、结论与建议

### ANL-003（P0）
数据清洗过程必须记录去除的数据量、原因和过滤条件

### ANL-004（P1）
统计分析必须注明置信区间和显著性水平

### ANL-005（P1）
分析结果必须经过至少一人交叉验证
