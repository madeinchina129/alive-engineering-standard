# 数据库对象命名规范 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| DN-001 | 表名使用业务领域名词复数形式：users, orders | P0 | 是 |
| DN-002 | 字段名使用小写和下划线：created_at | P0 | 是 |
| DN-003 | 索引命名：idx_表名_字段名 | P1 | 是 |

## 详细说明

### DN-001（P0）
表名使用业务领域名词复数形式：users, orders

### DN-002（P0）
字段名使用小写和下划线：created_at

### DN-003（P1）
索引命名：idx_表名_字段名
