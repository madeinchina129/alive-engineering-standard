# 用户故事规范 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| US-001 | 用户故事必须遵循「As a... I want... So that...」格式 | P0 | 是 |
| US-002 | 每个用户故事必须有唯一的 ID | P0 | 是 |
| US-003 | 每个用户故事必须有验收标准 | P0 | 是 |

## 详细说明

### US-001（P0）
用户故事必须遵循「As a... I want... So that...」格式

### US-002（P0）
每个用户故事必须有唯一的 ID

### US-003（P0）
每个用户故事必须有验收标准
