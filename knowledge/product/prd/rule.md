# PRD 标准模板 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| PRD-001 | 每个需求必须有明确的用户故事（As a... I want... So that...） | P0 | 是 |
| PRD-002 | 必须包含验收标准（Acceptance Criteria） | P0 | 是 |
| PRD-003 | 必须包含成功指标和衡量方式 | P1 | 是 |
| PRD-004 | 非功能性需求必须单独列出 | P1 | 是 |

## 详细说明

### PRD-001（P0）
每个需求必须有明确的用户故事（As a... I want... So that...）

### PRD-002（P0）
必须包含验收标准（Acceptance Criteria）

### PRD-003（P1）
必须包含成功指标和衡量方式

### PRD-004（P1）
非功能性需求必须单独列出
