# 验收标准定义 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| AC-001 | 验收标准使用 Given-When-Then 格式 | P0 | 是 |
| AC-002 | 每个用户故事至少 3 条验收标准 | P1 | 是 |
| AC-003 | 验收标准必须是原子性的 | P1 | 是 |

## 详细说明

### AC-001（P0）
验收标准使用 Given-When-Then 格式

### AC-002（P1）
每个用户故事至少 3 条验收标准

### AC-003（P1）
验收标准必须是原子性的
