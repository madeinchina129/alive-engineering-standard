# OWASP Top 10 防护 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| SEC-OWASP-001 | 所有 Web 应用必须实施 OWASP Top 10 防护措施 | P0 | 是 |
| SEC-OWASP-002 | 用户输入必须经过严格的输入验证和输出编码 | P0 | 是 |
| SEC-OWASP-003 | 所有 API 必须实施速率限制和请求大小限制 | P0 | 是 |
| SEC-OWASP-004 | 安全头部（CSP、X-Frame-Options、HSTS 等）必须正确配置 | P0 | 是 |
| SEC-OWASP-005 | CI/CD 流水线中必须集成 SAST 和 SCA 安全扫描 | P0 | 是 |

## 详细说明

### SEC-OWASP-001（P0）
所有 Web 应用必须实施 OWASP Top 10 防护措施

### SEC-OWASP-002（P0）
用户输入必须经过严格的输入验证和输出编码

### SEC-OWASP-003（P0）
所有 API 必须实施速率限制和请求大小限制

### SEC-OWASP-004（P0）
安全头部（CSP、X-Frame-Options、HSTS 等）必须正确配置

### SEC-OWASP-005（P0）
CI/CD 流水线中必须集成 SAST 和 SCA 安全扫描
