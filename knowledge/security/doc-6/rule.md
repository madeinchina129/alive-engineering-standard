# 安全漏洞管理 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| SEC-VULN-001 | 漏洞来源包括：SAST 扫描、SCA 扫描、DAST 扫描、渗透测试、安全公告 | P0 | 是 |
| SEC-VULN-002 | 漏洞按 CVSS 评分分级：高危（>= 7.0）、中危（4.0-6.9）、低危（< 4.0） | P0 | 是 |
| SEC-VULN-003 | 高危漏洞 7 天内修复，中危 30 天，低危 90 天 | P0 | 是 |
| SEC-VULN-004 | 修复完成后必须重新扫描验证漏洞已修复 | P0 | 是 |
| SEC-VULN-005 | 漏洞扫描工具每周至少执行一次全量扫描 | P1 | 推荐 |

## 详细说明

### SEC-VULN-001（P0）
漏洞来源包括：SAST 扫描、SCA 扫描、DAST 扫描、渗透测试、安全公告

### SEC-VULN-002（P0）
漏洞按 CVSS 评分分级：高危（>= 7.0）、中危（4.0-6.9）、低危（< 4.0）

### SEC-VULN-003（P0）
高危漏洞 7 天内修复，中危 30 天，低危 90 天

### SEC-VULN-004（P0）
修复完成后必须重新扫描验证漏洞已修复

### SEC-VULN-005（P1）
漏洞扫描工具每周至少执行一次全量扫描
