# Flutter 项目结构规范 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| FS-001 | 按 Feature 组织目录，每个包含 pages/widgets/providers/repositories | P0 | 是 |
| FS-002 | 共享代码放在 core/ 下：core/theme/, core/routes/, core/widgets/ | P1 | 是 |

## 详细说明

### FS-001（P0）
按 Feature 组织目录，每个包含 pages/widgets/providers/repositories

### FS-002（P1）
共享代码放在 core/ 下：core/theme/, core/routes/, core/widgets/
