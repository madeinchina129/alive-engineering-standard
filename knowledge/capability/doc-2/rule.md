# 业务能力映射 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| CAP-MAP-001 | 每个 L1/L2 能力必须至少映射到一个支撑系统 | P0 | 是 |
| CAP-MAP-002 | 映射关系必须标注支撑程度：强支撑（核心系统）、中支撑（辅助系统）、弱支撑（手工流程） | P0 | 是 |
| CAP-MAP-003 | 能力到应用系统的映射矩阵每季度审查一次 | P1 | 是 |

## 详细说明

### CAP-MAP-001（P0）
每个 L1/L2 能力必须至少映射到一个支撑系统

### CAP-MAP-002（P0）
映射关系必须标注支撑程度：强支撑（核心系统）、中支撑（辅助系统）、弱支撑（手工流程）

### CAP-MAP-003（P1）
能力到应用系统的映射矩阵每季度审查一次
