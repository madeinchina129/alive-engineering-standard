# 能力成熟度评估 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| CAP-MAT-001 | 评估维度包括：流程标准化、系统支撑度、自动化率、效率指标 | P0 | 是 |
| CAP-MAT-002 | 成熟度分 5 级：初始级、已管理级、已定义级、量化管理级、优化级 | P0 | 是 |
| CAP-MAT-003 | 每次评估必须有证据支撑（数据、文档、系统截图） | P0 | 是 |
| CAP-MAT-004 | 评估结果必须输出差距分析和改进路线图 | P0 | 是 |

## 详细说明

### CAP-MAT-001（P0）
评估维度包括：流程标准化、系统支撑度、自动化率、效率指标

### CAP-MAT-002（P0）
成熟度分 5 级：初始级、已管理级、已定义级、量化管理级、优化级

### CAP-MAT-003（P0）
每次评估必须有证据支撑（数据、文档、系统截图）

### CAP-MAT-004（P0）
评估结果必须输出差距分析和改进路线图
