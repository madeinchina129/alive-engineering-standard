# CQRS 模式 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| EDA-CQRS-001 | 命令必须使用领域模型处理，包含完整的业务验证 | P0 | 是 |
| EDA-CQRS-002 | 查询模型必须从事件投影构建，不能直接操作命令数据库 | P0 | 是 |
| EDA-CQRS-003 | 查询模型可以包含反范式化数据以优化查询性能 | P0 | 是 |
| EDA-CQRS-004 | 读库和写库的数据同步延迟必须在 SLA 范围内 | P1 | 推荐 |

## 详细说明

### EDA-CQRS-001（P0）
命令必须使用领域模型处理，包含完整的业务验证

### EDA-CQRS-002（P0）
查询模型必须从事件投影构建，不能直接操作命令数据库

### EDA-CQRS-003（P0）
查询模型可以包含反范式化数据以优化查询性能

### EDA-CQRS-004（P1）
读库和写库的数据同步延迟必须在 SLA 范围内
