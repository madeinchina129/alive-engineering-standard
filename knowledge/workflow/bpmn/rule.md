# BPMN 标准实践 — 规则

## 规则列表

| 规则 | 说明 | 优先级 | 强制 |
|------|------|--------|------|
| WF-BPMN-001 | BPMN 模型必须遵循 BPMN 2.0 规范，使用标准符号 | P0 | 是 |
| WF-BPMN-002 | 流程模型按三级分层：流程域 → 端到端流程 → 子流程 | P0 | 是 |
| WF-BPMN-003 | 泳道（Lane）按角色/部门划分，清晰展示职责分工 | P0 | 是 |
| WF-BPMN-004 | 排他网关必须有默认分支，避免流程卡死 | P0 | 是 |
| WF-BPMN-005 | 消息流用于跨组织/跨系统通信，与顺序流区分 | P1 | 推荐 |

## 详细说明

### WF-BPMN-001（P0）
BPMN 模型必须遵循 BPMN 2.0 规范，使用标准符号

### WF-BPMN-002（P0）
流程模型按三级分层：流程域 → 端到端流程 → 子流程

### WF-BPMN-003（P0）
泳道（Lane）按角色/部门划分，清晰展示职责分工

### WF-BPMN-004（P0）
排他网关必须有默认分支，避免流程卡死

### WF-BPMN-005（P1）
消息流用于跨组织/跨系统通信，与顺序流区分
